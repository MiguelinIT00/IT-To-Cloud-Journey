# AWS Fundamentals Notes

## What AWS is

AWS stands for Amazon Web Services. It is a cloud platform that provides services like computing, storage, networking, security, and databases.

## Why cloud matters

Cloud platforms let companies run infrastructure without buying and maintaining everything on-premises.

## Core ideas I'm learning

- On-demand resources

- Scalability

- Pay-as-you-go pricing

- Global infrastructure

- Shared responsibility model

## Services I'm focusing on first

- EC2 for virtual servers

- S3 for storage

- IAM for access control

- VPC for networking basics

Learn enough to understand how cloud environments are built, managed, and secured so I can grow from IT support into cloud-focused roles.
