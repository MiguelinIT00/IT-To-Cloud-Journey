# EC2 Basics (Elastic Compute Cloud)
## What EC2 is
Amazon EC2 (Elastic Compute Cloud) allows you to run virtual servers in the cloud.
Instead of buying physical hardware, you can launch and manage servers on demand.
## Why EC2 matters
EC2 is one of the core AWS services. It is used to:
- Host applications
- Run backend services
- Test environments
- Replace on-prem servers
## Key concepts
### Instance
A virtual server running in AWS.
### AMI (Amazon Machine Image)
A template used to launch an instance (like choosing Windows or Linux).
### Instance Type
Defines the size and power of the server (CPU, RAM, etc).
### Key Pair
Used to securely connect to your instance (SSH for Linux).
### Security Groups
Acts like a firewall controlling inbound and outbound traffic.
