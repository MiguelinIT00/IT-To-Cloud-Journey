# EC2 First Launch Lab
## Objective
Launch my first EC2 instance and understand the setup process.
## Steps I followed
- Logged into AWS Management Console
- Navigated to EC2 Dashboard
- Clicked "Launch Instance"
- Selected an AMI (Amazon Linux / Windows)
- Chose instance type (t2.micro for free tier)
- Created or selected a key pair
- Configured security group (allowed SSH)
- Launched the instance
## What I learned
- EC2 setup involves multiple configuration steps
- Security groups are critical for access control
- Key pairs are required for secure login
- Instances can be started, stopped, and terminated easily
## Challenges
- Understanding networking and ports
- Knowing which settings are safe vs risky
